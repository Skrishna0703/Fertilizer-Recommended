║                          TROUBLESHOOTING                                     ║
╚══════════════════════════════════════════════════════════════════════════════╝

❓ "Images not showing in results"
  → Refresh browser (Ctrl+R)
  → Check images are in: images/Urea, images/DAP, etc.
  → Verify file names end with .jpg, .png, .jpeg, or .webp

❓ "App won't start"
  → Make sure you're in Project\Fertilizer directory
  → Run: streamlit run app.py
  → Visit: http://localhost:8501

❓ "Model not predicting"
  → Run train_model.py first: python train_model.py
  → Verify .pkl files were created
  → Check input values are valid (N/P/K 0-100, pH 5-9)

❓ "Placeholder showing for images"
  → Folder exists but image missing
  → Check folder path shows in placeholder
  → Add image to correct folder
  → Refresh browser

For more help, see IMAGES_README.md or COMPLETE_SETUP.md
