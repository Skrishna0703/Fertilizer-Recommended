## 🎓 Tips

### Best Practice:
- Use **real fertilizer bag photos** (not generic agriculture images)
- Show **product packaging** (bags/containers)
- Use **consistent image quality** across all fertilizers
- Include **N-P-K ratio visible** in the image (e.g., "46-0-0", "10-26-26")

### Recommended Image Content:
- **Urea:** White/off-white granules, bag showing 46% N
- **DAP:** Gray/brown granules, showing 18:46 ratio
- **NPK Balanced:** Mixed colors, showing balanced ratio like 10:26:26
- **SSP:** Grayish powder/granules, showing phosphorus percentage

## 🔧 Troubleshooting

**Q: Images still not showing?**
A: 
1. Ensure image is in the CORRECT folder name: `Urea`, `DAP`, `NPK_Balanced`, `SSP`
2. Check file extension: must be `.jpg`, `.png`, `.jpeg`, or `.webp`
3. Restart Streamlit app completely
4. Refresh browser (Ctrl+R)

**Q: Which image shows if I add multiple?**
A: First image alphabetically. Example: `001_urea_bag.jpg` displays first

**Q: Can I use images from internet?**
A: Yes! But ensure you have rights to use them

**Q: Image quality looks bad?**
A: Try a different image with better resolution
